## 起步

``` bash
# 安装依赖
npm install

# 运行服务，端口可在config目录下index.js文件配置
npm run dev

```

### 可能的问题
运行过程中如果发现请求错误，请大家自行修改一下请求地址，位置如下：

 ```/src/api/api.js```
